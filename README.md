
# 🍅 Tomato Leaf Disease Prediction

## 📝 About The Project
Welcome to my final project for the Applied Machine Learning course! 

Plant diseases are a huge problem in agriculture, especially for tomato crops. I built this deep learning project to help automatically identify whether a tomato leaf is healthy or if it has a disease. I trained a Convolutional Neural Network (CNN) using TensorFlow to classify the leaves into 10 different categories. To make it easy for anyone to use, I also created a simple web interface using Gradio.

This project was developed as part of the Digital Skills for Students Program under the EDGE Project.

## ✨ Features
* **Image Upload:** Users can easily upload a picture of a tomato leaf.
* **Fast Prediction:** The AI quickly analyzes the image and identifies the disease.
* **Confidence Score:** It shows a percentage (like 98% Early Blight) so you know how confident the model is.
* **User-Friendly Dashboard:** A clean, green-themed GUI built with Gradio.

## 🛠️ Tech Stack
* **Language:** Python
* **Deep Learning Framework:** TensorFlow / Keras
* **Web GUI:** Gradio
* **Data Processing:** NumPy, Pillow (PIL)

## 📊 Dataset
The model was trained on the publicly available PlantVillage dataset. It can identify the following 10 classes:
1. Bacterial Spot
2. Early Blight
3. Late Blight
4. Leaf Mold
5. Septoria Leaf Spot
6. Spider Mites
7. Target Spot
8. Yellow Leaf Curl Virus
9. Mosaic Virus
10. Healthy
